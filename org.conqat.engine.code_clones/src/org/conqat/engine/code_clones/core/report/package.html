The classes in this package read and write clone reports. They are intended for use inside and outside of ConQAT. Hence, they operate with lists of {@link org.conqat.engine.code_clones.core.report.SourceFileDescriptor}s and {@link org.conqat.engine.code_clones.core.CloneClass}es, instead of with {@link org.conqat.engine.code_clones.detection.CloneDetectionResultElement}s.

The {@link org.conqat.engine.code_clones.result.CloneReportReaderProcessor} and the {@link org.conqat.engine.code_clones.result.CloneReportWriterProcessor} perform the adaption between these lists and {@link org.conqat.engine.code_clones.detection.CloneDetectionResultElement}s.
