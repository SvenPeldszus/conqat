This bundle contains processors which allow the definition of new processors from within the ConQAT configuration. For this the Java-like scripting language BeenShell it made available through the {@processor edu.tum.cs.conqat.scripting.BeanShellProcessor}. For an example see the block {@block edu.tum.cs.conqat.scripting.BshExample}. Additionally there are more lightweight processors for performing general arithmetic operations.
